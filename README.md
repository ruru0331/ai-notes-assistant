# AI Notes Assistant

AI Notes Assistant is a simple web app that helps organize messy notes into a summary, keywords, and action items.

This project is built as a beginner-friendly AI-style productivity tool using HTML, CSS, and JavaScript.

## Features

- Summarize pasted notes
- Extract simple keywords
- Detect possible action items
- Try a sample note with one click
- Clear the workspace and start again
- Run directly in the browser

## Live Demo

Visit the live site here:

https://ruru0331.github.io/ai-notes-assistant/

## How to Use

1. Paste notes, meeting ideas, or study material into the text box.
2. Click `Analyze Notes`.
3. Review the generated summary, keywords, and action items.
4. Use `Try sample` to test the app quickly.
5. Use `Clear` to reset the page.

## Project Status

This is an early version of the project.

The current version uses basic JavaScript logic to simulate AI-style note organization. Future versions may connect to a real AI API or support uploaded documents.
